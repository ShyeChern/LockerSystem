------------------------------------------------
Things to note and step to do to run the project
------------------------------------------------

1. Both devices must connect to same internet connection
2. Change the ip address in startServer method according to your connection in LockerController.java
3. Change the ip address in startClient method according to the ip address of startServer method
4. Run LockerServer.java to startServer (for Server)
5. Run Register.java (for Client)
6. Everything is done and you can start to demo the project
